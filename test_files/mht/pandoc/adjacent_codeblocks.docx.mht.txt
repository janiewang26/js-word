We can identify this interface by listing the contents of the /sys/class/ieee80211 directory, as shown.	# ls /sys/class/ieee80211/phy0The "phy0" interface is the parent interface used to create child interfaces. Note that if you unplug and replug the USB interface, the "phy" interface number will increment by one until you reboot your system.
